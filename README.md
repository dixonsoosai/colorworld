# colorworld
Billing Software for Retail Business

#Project Setup

Pre-requisties:
1) OpenJdk17
2) Postgresql with server
3) Eclipse
4) Visual Studio
5) Nodejs 18.16.0

Steps:
1) File -> Import -> Projects from Git
2) Enter all the details viz. URI, username & password
3) Select master branch
4) Import
5) mvn clean install

#Angular
npm install
npm start
